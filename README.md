# 📋 DID 게시물 신청 시스템

이 웹앱은 **디지털 게시판(DID) 게시물의 게시 기간을 관리하고 시각화하는 시스템**입니다.

---

## 🎯 핵심 기능

### 1. 게시물 등록 기능

**입력 항목:**
- 부서 (필수)
- 담당자 (필수)
- 제목 (필수)
- 게시 시작일 (필수)
- 게시 종료일 (필수)
- 파일 첨부 (선택, 1개, 최대 10MB)

### 2. 게시 기간 차트 (Gantt Chart)

**시각화 기능:**
- 각 게시물의 게시 기간을 간트 차트로 표시
- 오늘 날짜를 빨간색 수직선으로 표시
- 드래그 앤 드롭으로 순서 변경 가능

**게시 상태별 색상:**
- 🟢 **게시 중** (active): 녹색 그라데이션
- 🔵 **게시 예정** (pending): 파란색 그라데이션
- ⚫ **게시 종료** (expired): 회색 그라데이션

### 3. 게시물 목록 관리

**테이블 형식 표시:**
- 부서, 담당자, 제목, 시작일, 종료일, 상태, 첨부파일, 수정/삭제 버튼

**정렬 기능:**
- 시작일 기준 오름차순/내림차순
- 종료일 기준 오름차순/내림차순

**추가 기능:**
- 드래그 앤 드롭으로 항목 순서 변경
- 게시 상태 실시간 표시 (게시 중/예정/종료)
- 아이콘 버튼 (연필=수정, 휴지통=삭제)

### 4. 게시물 수정 기능

- **수정 모달:** 게시물 클릭 시 편집 가능한 모달 창 표시
- **수정 가능 항목:** 부서, 담당자, 제목, 시작일, 종료일, 첨부파일
- **파일 관리:**
  - 기존 파일 유지 또는 삭제 가능
  - 새로운 파일로 교체 가능
  - 파일 추가 (기존에 없던 경우)
- **저장 중 표시:** 저장 버튼이 "저장 중..."으로 변경되어 진행 상황 표시
- **이메일 알림:** 게시물 수정 시 관리자에게 이메일 발송

### 5. 파일 관리

- **첨부파일 다운로드:** 
  - 모든 파일 타입 강제 다운로드 (브라우저에서 열리지 않음)
  - 이미지 파일도 컴퓨터에 직접 저장
- **파일 크기 제한:** 최대 10MB
- **파일 개수 제한:** 1개
- **자동 삭제:** 게시물 삭제 시 첨부파일도 Firebase Storage에서 자동 삭제

### 6. 삭제 기능

- **개별 삭제:** 각 게시물마다 휴지통 아이콘 버튼 제공
- **전체 삭제:** "전체 삭제" 버튼으로 모든 게시물 일괄 삭제
- **이메일 알림:** 게시물 삭제 시에도 관리자에게 이메일 발송
- **확인 메시지:** 삭제 전 확인 팝업 표시

## 📱 사용자 경험 특징

- ✅ 간단한 폼 기반 입력
- ✅ 시각적인 간트 차트로 게시 기간 확인
- ✅ 실시간 게시 상태 표시 (게시 중/예정/종료)
- ✅ 드래그 앤 드롭으로 직관적인 순서 변경
- ✅ 모바일에서도 사용 가능한 반응형 디자인
- ✅ 자동 이메일 알림으로 변경사항 추적
- ✅ 게시물 수정 기능으로 유연한 관리
- ✅ 저장 중 표시로 명확한 피드백
- ✅ 모든 파일 타입 강제 다운로드 지원
